# TY Multilog

Generate geospatial data about active tropical cyclone in the Western North Pacific

Data are retrieved from the following websites:
1. [RAMMB](http://rammb-data.cira.colostate.edu/tc_realtime/)
2. [JTWC](https://www.metoc.navy.mil/jtwc/jtwc.html)
3. [Typhoon2000](http://www.typhoon2000.ph/)
